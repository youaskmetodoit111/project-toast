Information about Mesh Linux:

Mesh Linux is a linux distro designed for laptops, desktops and x86 tablets. it does NOT matter if it has either a sim card slot or not. It is actually based on Debian but there are features: Perfect XFCE4 desktop, chromium web browser, startup sound and etc. Chromium Web Browser - You can switch between Firefox and Chromium. Chromium is optional to access all websites.

Features in v0.9

- We ditched Phosh, GNOME in discontinued versions like kwoka which is 0.7 and 0.8 toast which are removed from the repositories and replaced it with XFCE4 which makes it super easy to make our new wallpaper as default

Features in v1.0

- Optimized for touchscreen tablets, touchscreen laptops, non-touchscreen laptops and desktop computers - Onboard pre-installed onto this syste
