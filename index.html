Information about Mesh Linux:

Mesh Linux is a linux distro designed for laptops specifically the purism librem 14. it does NOT matter if it has either a sim card slot or not. It is actually based on Debian but there are features: Dualbooting between GNOME and Phosh on the lock screen. Chromium Web Browser - You can switch between Firefox and Chromium. Chromium is optional to access all websites.

Features in v0.7:

- Chromium - Switch between Phosh and GNOME - Phosh desktop enviorement

Features in v0.8

- New background (to set it as default, open the file manager, at the top search /usr/share/backgrounds/gnome/ and once you click enter look for a file named ugol2.png just by scrolling, right click it and then look for an option and click set as wallpaper and you're done!

Features in v0.9

- We ditched Phosh, GNOME and replaced it with XFCE4 which makes it super easy to make our new wallpaper as default

Features in v1.0

- Optimized for touchscreen tablets, touchscreen laptops, non-touchscreen laptops and desktop computers - Onboard pre-installed onto this syste
